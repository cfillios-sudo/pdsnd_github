### BikeShare Data Analysis Project
This repository captures code support the Bikeshare Data Analysis created in support of Udacity training program. 

### Description
This project analyzes bikeshare usage data for three major U.S. cities: Chicago, New York City, and Washington. The Python program allows users to interactively explore bikeshare data by selecting a city, month, and day of the week, and then displays relevant usage statistics.

The purpose of this project is to demonstrate basic Python programming skills, including data analysis with pandas, user input handling, and the use of version control with Git and GitHub.

### Files used
- bikeshare.py — Main Python script used to analyze bikeshare data
- chicago.csv — Bikeshare data for Chicago
- new_york_city.csv — Bikeshare data for New York City
- washington.csv — Bikeshare data for Washington

Note: Data files are excluded from version control using .gitignore.

### Software Requirements
- Python 3.x
- pandas
- NumPy (optional but recommended)

### Statistics Computed
The program computes and displays the following statistics:
- Time Statistics (most common month, day, or hour of travel)
- Station Statistics (most common start, end, and combination of start and end stations)
- Trip Duration Statistics (total and average travel time)
- User Statistics (when available) (count of user types, gender, and years of birth)
